I want browsers to provide a native API for GDPR consent so that each website no longer has to implement its own pop-up, modal, or overlay to collect user consent.

Today, every site I visit greets me with a different cookie banner — each with its own design, placement, and dark patterns. The experience is inconsistent, often confusing, and almost universally annoying. Browsers already know my locale and could reasonably know my privacy preferences; they should expose a standardised API that lets sites query those preferences without repeatedly interrupting me.

A browser-level consent API would make the experience both more consistent and less obtrusive across sites, while giving users a single, trusted place to manage their privacy choices rather than negotiating with hundreds of individual websites.
